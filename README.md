## Sample Listing Application

- Getting fake data from [fakerapi](https://fakerapi.it/en)
- React app generated using Vite
- Typescript for type safety

## Features

1. Listing Data in a Table format
2. Sorting for certain fields is provided
3. Number of Rows of data to get can be controlled
4. Table component is written as generic and can be used with any kind of other data
5. User is let know of loading and errors
6. Rows can be expanded to view the additional details

## Development

Make sure you have node and yarn installed and run the command

```bash
yarn install
yarn run dev
```

## Production

To create a production build you can use

```bash
yarn run build
```
